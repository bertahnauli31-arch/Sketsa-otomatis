# 🖌️ Auto Paint Drawer

Program ini memungkinkan komputer menggambar otomatis di aplikasi **Paint** berdasarkan gambar yang dimuat.  
Cukup **load gambar**, arahkan kursor ke kanvas Paint, lalu tekan **Start** → kursor akan bergerak mengikuti pola gambar dan menggambar secara otomatis.

---

## ✨ Fitur Utama
- Load gambar dari file (`.jpg`, `.png`, dll).
- Konversi gambar ke koordinat pixel.
- Kontrol kursor otomatis menggunakan `pyautogui`.
- Simulasi klik/drag untuk menggambar di Paint.
- Bisa dioptimasi untuk menggambar outline atau full pixel.

---

## ⚙️ Teknologi
- **Python 3.x**
- **PIL / OpenCV** → pemrosesan gambar
- **PyAutoGUI** → kontrol mouse
